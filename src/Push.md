- ### `git push` sends these changes to GitHub so others can see or work on them

`git push <remote> <branch>`
- `remote`usually origin (default remote repository).

- `branch`the branch you want to push to,` main `or `master`

`git push -u origin main`
- `-u` sets this branch as the default for future pushes.

- Next time, you can simply run `git push`main
